# 操作マニュアル 3分冊化計画

## 概要

既存のアニメーション付きモックアップ形式を活かし、以下の **3つの独立した操作マニュアル** を作成する。

| # | マニュアル | URL | 対象 | ステータス |
|---|----------|-----|------|-----------|
| 1 | 会員向け操作マニュアル | `/manual` | 住民 | ✅ 既存（変更なし） |
| 2 | 役員管理画面 操作マニュアル | `/manual/admin` | 役員 | 🆕 新規作成 |
| 3 | Stripe連携 操作マニュアル | `/manual/stripe` | 役員 | 🆕 新規作成 |

変更なし。現在の10ステップをそのまま維持。

## 2. 役員管理画面 操作マニュアル（新規 `/manual/admin`）

#### [NEW] [page.tsx](file:///C:/Users/info/.gemini/antigravity/scratch/qoin/src/app/manual/admin/page.tsx)

**10ステップ** の管理画面操作ガイド：

| ステップ | タイトル | 模型の表示内容 |
|---------|---------|--------------|
| **1** | LINEリッチメニューから管理画面を開く | LINEトーク → 「役員の方」ボタン |
| **2** | ダッシュボードの見方 | 統計カード（未読率・プッシュ数・会費状況） |
| **3** | 回覧板・お知らせを作成する | 新規作成モーダル（タイトル・本文・カテゴリ） |
| **4** | プッシュ通知を送信する | プッシュ送信ON/OFF＋配信確認 |
| **5** | イベントの出欠を確認する | 申込者一覧の表示 |
| **6** | 会員名簿を管理する | 名簿一覧・追加・CSV登録 |
| **7** | 会費の管理・集金状況を確認する | 会費管理タブ（完納/未納ステータス） |
| **8** | 町内会・自治会の基本設定 | 設定画面（名称・郵便番号・管理者情報） |
| **9** | 管理者（役員）を追加する | 管理者設定タブ |
| **10** | 画面の閉じ方 | × ボタンでLINEに戻る |

## 3. Stripe連携 操作マニュアル（新規 `/manual/stripe`）

#### [NEW] [page.tsx](file:///C:/Users/info/.gemini/antigravity/scratch/qoin/src/app/manual/stripe/page.tsx)

**8ステップ** のStripe連携ガイド：

| ステップ | タイトル | 模型の表示内容 |
|---------|---------|--------------|
| **1** | Stripe決済とは？ | メリット一覧（初期費用0円・手数料3.6%・セキュリティ） |
| **2** | テスト連携を開始する | 設定 → オンライン集金 → 「連携する」ボタン |
| **3** | Stripe登録画面の入力ポイント | 事業形態・代表者情報・銀行口座の入力画面 |
| **4** | テスト連携の完了 | 「Stripe連携が完了しています」画面 |
| **5** | テスト決済で動作確認 | テスト用カード（4242...）での支払い画面 |
| **6** | 本番への切替（リセット） | 「解除する」ボタン → 再連携の流れ |
| **7** | 本番アカウントの登録 | 本人確認書類・実口座情報の入力 |
| **8** | 本番連携の完了＆審査 | 完了画面 + 審査1〜2営業日の案内 |

## 4. 管理画面の簡素化

#### [MODIFY] [AdminView.tsx](file:///C:/Users/info/.gemini/antigravity/scratch/qoin/src/components/AdminView.tsx)

Stripeセクション（オンライン集金タブ）を以下に簡素化：

**削除する部分**：
- テスト/本番切替ガイダンス（青いボックス）
- 本番環境への登録ガイダンス（緑のボックス）
- 準備するものチェックリスト（黄色のボックス）
- 3大解説ガイド アコーディオン 3つ（①推奨理由 ②任意団体ガイド ③テスト手順）

**残す部分**：
- 簡潔な1行説明
- 「Stripeアカウントを登録・連携する」ボタン
- 「Stripe連携が完了しています」バッジ + アカウントID
- 「Stripe連携を解除（リセット）する」ボタン

**追加する部分**：
- 📖 **操作マニュアルへのリンクボタン**（`/manual/stripe` を新しいタブで開く）
