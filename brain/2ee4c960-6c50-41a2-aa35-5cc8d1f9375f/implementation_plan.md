# 総会管理機能（予算書・決算書）および住民出欠返信機能の実装

## 背景

町内会・自治会の総会運営をデジタル化するため、以下の機能を実装します。
1. **管理者向け**: 年度ごとの科目設定、予算作成、決算実績入力、予算書・決算書（PDF/CSV）の出力機能（すでに `scratch/` に下書きコード作成済み）。
2. **住民向け**: 総会案内の閲覧、オンラインでの出欠回答（出席・欠席）、および欠席時の委任状ファイル（画像・PDF）のアップロード提出機能。

## データベース (Supabase)

### [NEW] [Day51_create_assembly_tables.sql](file:///c:/Users/info/.gemini/antigravity/scratch/qoin/database/Day51_create_assembly_tables.sql)

総会予算・決算機能に必要な以下の3つのテーブルを新規作成します。

```sql
-- 1. 総会用 科目カテゴリテーブル
CREATE TABLE IF NOT EXISTS public.assembly_categories (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    neighborhood_id BIGINT NOT NULL REFERENCES public.neighborhoods(id) ON DELETE CASCADE,
    type TEXT NOT NULL CHECK (type IN ('income', 'expense')),
    name TEXT NOT NULL,
    parent_id BIGINT REFERENCES public.assembly_categories(id) ON DELETE CASCADE,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- 2. 総会用 予算テーブル
CREATE TABLE IF NOT EXISTS public.assembly_budgets (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    neighborhood_id BIGINT NOT NULL REFERENCES public.neighborhoods(id) ON DELETE CASCADE,
    fiscal_year INT NOT NULL,
    category_id BIGINT NOT NULL REFERENCES public.assembly_categories(id) ON DELETE CASCADE,
    budget_amount INT DEFAULT 0,
    previous_budget_amount INT DEFAULT 0,
    note TEXT,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
    UNIQUE(neighborhood_id, fiscal_year, category_id)
);

-- 3. 総会用 決算実績明細テーブル
CREATE TABLE IF NOT EXISTS public.assembly_settlements (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    neighborhood_id BIGINT NOT NULL REFERENCES public.neighborhoods(id) ON DELETE CASCADE,
    fiscal_year INT NOT NULL,
    category_id BIGINT NOT NULL REFERENCES public.assembly_categories(id) ON DELETE CASCADE,
    amount INT DEFAULT 0,
    paid_date DATE NOT NULL,
    description TEXT,
    receipt_url TEXT,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- RLSの有効化とパブリックアクセス許可（既存のテーブル設計方針と整合）
ALTER TABLE public.assembly_categories ENABLE ROW LEVEL SECURITY;
ALTER TABLE public.assembly_budgets ENABLE ROW LEVEL SECURITY;
ALTER TABLE public.assembly_settlements ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Enable all for authenticated users on assembly_categories" ON public.assembly_categories USING (true) WITH CHECK (true);
CREATE POLICY "Enable all for authenticated users on assembly_budgets" ON public.assembly_budgets USING (true) WITH CHECK (true);
CREATE POLICY "Enable all for authenticated users on assembly_settlements" ON public.assembly_settlements USING (true) WITH CHECK (true);
```

> [!IMPORTANT]
> このSQLは手動でSupabase of のSQL Editorで実行する必要があります。
