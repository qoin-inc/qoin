# [広域交流サイト（マイel-town）の構築プラン]

ご要望に沿って、画面構成やボタン配置を大きく見直した新しい実装プランを作成しました。

## Proposed Changes

### 1. リッチメニューのボタン追加と配置変更
#### [MODIFY] src/components/ResidentView.tsx
- ログイン済みの会員画面（リッチメニュー）にて、「設定・その他」タブの中にある**「役員の方へ」ボタンの横に「マイel-town」ボタン**を追加します。
- このボタンをタップすると、全国交流ポータル（`/portal`）へ移動します。

### 2. 交流ポータル画面の3機能への分割
広域ポータル画面のデザインを一新し、以下の3つのボタン（タブ）で画面を切り替えるようにします。

#### [MODIFY] src/app/portal/page.tsx
- **① 食べel-town:** 
  全国の美味しいお店などの投稿が流れる「タイムライン（一覧表示）」の画面です。
- **② 観えel-town:** 
  全国のイベント情報が一覧できる「専用のイベントカレンダー表示」の画面です。
- **③ マイel-town:** 
  「全面地図」の画面です。地図上には、①②で情報を発信した町内会・自治会のピンが立ちます。
  - **ピンからのリンク:** ピンをタップすると、その町内会から発信された情報（食べ・観えの最新投稿）が表示され、さらに詳しい情報へリンク（ジャンプ）できるようにします。

### 3. 発信時の自動ピン反映（郵便番号連携）
#### [MODIFY] src/app/portal/page.tsx
- 会員が①や②で情報を発信した際、システムが自動的にその会員の町内会の「郵便番号」を拾い上げ、緯度・経度を取得して地図（③）へピンを反映する処理を組み込みます。
