# 複数管理者機能（最大10名）の実装完了

町内会の「管理者」を最大10名まで登録でき、かつ個別に削除（権限剥奪）できる機能を実装しました。
また、懸念されていた「それぞれの役員が自分でパスワードを決める」運用も「招待システムの導入」により解決しています。

## ⚠️ ご確認の前に必ず行っていただくこと
新しいデータベース構造（一覧テーブル）を作りましたので、画面を動かす前に以下のSQLスクリプトを実行してください。これを実行しない場合、ログインができなくなります。

1. **`database/Day10_multiple_admins.sql`** の中身をコピーします。
2. Supabase の SQL Editor に貼り付けて、`Run` ボタンを押して実行してください。
   *※これまでに登録されていた初期代表者やテストデータの情報も自動的に新しいテーブルへ引き継がれます。*

---

## 新しい機能のテスト手順（使い方）

### 1. 新しい役員の「招待」
1. いままでのテスト用アカウントで、通常通り **[管理機能へログイン]** します。
2. 画面右上の **[設定] アイコン** をクリックします。
3. 一番右に新しく増えた **[管理者設定] タブ** をクリックします。
4. すでにあなたの名前が「連携済」として表示されており、右のゴミ箱で自分自身も削除（退出）できる権限を持っていることが確認できます。
5. 「新しい管理者を招待する」から、テスト用の適当な名前とメールアドレス（例: `sub@example.com` など）を入力して **[招待リストに追加]** を押します。
6. リストに追加され、ステータスが **「招待中」** になります。

### 2. 新しい役員の「合流（登録）」
1. ポータル画面に戻り、再び「el-town管理機能」の入口を開きます。（一度ログアウトしてください）
2. ログイン画面の少し下にある **[代表者から招待された方はこちら]** というオレンジ色のボタンを押します。
3. さきほど招待リストに追加したメールアドレス（例: `sub@example.com`）を入力し、**ご自身のお好きなパスワード**を設定して「合流する」ボタンを押します。
4. データベース上で照合され、即座に同じ町内会のダッシュボードに入れるようになります！
5. （もう一度「設定」の「管理者設定」を覗くと、その人のステータスが「連携済」に切り替わっています）

### 3. 不要になった役員の「削除」
設定画面で、不要になった役員の右側にある **[ゴミ箱ボタン]** を押してください。
確認画面でOKを押せば、即座に権限が剥奪され、その方はログインできなくなります。
